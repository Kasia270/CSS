Co znaczy skrót ID? ID ID oznacza identyfikator.
Jaki jest cel stosowania ID? ID pozwala na osobne różnicowanie stylu w stosunku do innych podobnych elementów.
Jakie jest znaczenie znaku # Znak # w arkuszu CSS oznacza, że odwołujemy się do elementu po jego identyfikatorze (ID), np. #pochyly.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Wywołanie następuje w HTML poprzez id="nazwa", np.:
id="pochyly">Tekst

W CSS używamy znaku #:
p#pochyly { font-style: italic; }
Uwagi na temat stosowania ID (trzy konieczne wyliczenia)
  1. ID powinien być użyty tylko jeden raz w całym dokumencie.
  2. Wielokrotne użycie tego samego ID jest błędem.
  3. Jeśli nie mamy pewności, że element wystąpi tylko raz, lepiej użyć class.
